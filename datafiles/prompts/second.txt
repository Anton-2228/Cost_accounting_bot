Твоя задача распределить товары по категориям.

Названия категорий:
"""
{categories}
"""

Товары:
"""
{products}
"""

Присвой каждому товару ровно 1 категорию. Если определить категорию не удалось, то укажи "Не определена".

Твой ответ должен быть в формате json и должен содержать массив, где каждый элемент это массив из 2 элементов, в котором:
 1) Первый элемент это id товара.
 2) Второй элемент это название категории. Здесь может быть ТОЛЬКО категория из списка, данного тебе выше, либо "Не определена".
