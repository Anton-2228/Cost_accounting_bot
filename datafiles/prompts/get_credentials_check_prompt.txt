Твоя задача достать из чека эти данные:
 1) ФН - число
 2) ФД - число
 3) ФП - число
 4) Дата - дата и время через пробел
 5) Итого - дробное число

Твой ответ должен быть в формате json с 3 полями:
 1) fn - в этом поле значение ФН
 2) fd - в этом поле значение ФД
 3) fp - в этом поле значение ФП
 4) t - в этом поле значение Дата
 5) s - в этом поле значение Итого
