Твоя задача извлечь из чека товары и присвоить каждому из них тип из данного тебе списка.

Текст чека:
"""
{check}
"""

Типы через запятую:
"""
{types}
"""

Присвой каждому товару ровно 1 тип. Если в списке выше нет типа, подходящего под товар, то предложи новый.

Твой ответ должен быть в формате json и должен содержать массив, где каждый элемент это массив из 4 элементов, в котором:
 1) Первый элемент это название товара
 2) Второй элемент это его цена. Без указания валюты. Только число
 3) Третий элемент это данный ему тип. Здесь может быть ТОЛЬКО тип из данного тебе списка типов. Если в списке нет подходящего типа, то здесь должна быть пустая строка.
 4) Четвертый элемент это тип, который ты можешь предложить. Если в данном тебе списке типов найден подходящий тип, то здесь должна быть пустая строка.
