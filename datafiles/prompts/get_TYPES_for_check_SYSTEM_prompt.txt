Твоя задача присвоить каждому товару тип из данного тебе списка, либо предложить новый.

Присвой каждому товару ровно 1 тип. Если в списке выше нет типа, подходящего под товар, то предложи новый.

Твой ответ должен быть в формате json, аналогичном тому, в котором были переданы товары, но должен содержать 2 новых поля:
 1) type - в этом поле может быть ТОЛЬКО тип из данного тебе списка выше.
 2) new_type - в этом поле должен быть предложенный тип.
 ВАЖНО - заполнено может быть только 1 поле. Другое должно быть null.
